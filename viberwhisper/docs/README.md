# ViberWhisper 文档目录

本文档按功能分类组织，方便查找和维护。

## 目录结构

```
docs/
├── README.md              # 本文档
├── todo_new.md            # 当前 TODO 任务列表
├── features/              # 功能实现文档 (plan)
│   ├── 01-global-hotkey-recording.md
│   ├── toggle_recording_plan.md
│   ├── cross_platform_plan.md
│   ├── tray_icon_plan.md
│   └── logging_refactor_plan.md
├── architecture/          # 架构设计文档 (待补充)
└── guides/                # 使用指南 (待补充)
```

## 分类说明

### features/
功能实现计划文档，每个功能一个 plan 文件：
- 需求描述
- 技术选型
- 实现步骤
- 注意事项

### architecture/
架构设计文档：
- 模块设计
- 数据流图
- API 接口定义

### guides/
使用指南：
- 安装说明
- 配置教程
- 故障排查
